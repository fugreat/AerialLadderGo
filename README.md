# AerialLadderGo    
 我有一个云梯车，想把它改造成微信遥控的。就是这样。     
-------------------------------------  


##需求分解   
基本功能 |  细分功能   
----   | -----    
运动控制 |  前进/后退      
         |  左转/右转   
         |  吊臂伸缩   
         |  吊臂左右转    
声光报警 |  前灯   
         |  后尾灯   
         |  顶部警灯   
电量指示 |  电量采集   
         |  电量上传   
         |  电量显示    
wifi联网 |  wifi账号配置   
         |  wifi状态上传  
微信绑定 |  唯一码生成  
         |  账号绑定
